Per poder jugar a ScapePrison s'han de seguir els següents passos:

Descarregar el programa XAMPP.
https://www.apachefriends.org/es/download.html   
Start a Apache i MySQL. 

Anar a la web:
localhost/phpmyadmin/ (o localhost/phpMyAdmin/ a MAC)

Crear una nova base de dades anomenada: "BaseDeDatosJuego2"

Crear una taula amb 4 columnes que s'anomeni: "DatosJuego2"
Les 3 primeres posicions seràn Strings(opció: Text) i la última serà un Int que s'anomenaran:

"Mapa1":
***************************************************************************************F*******************F*****F*****F************************
"Mapa2":
*******************F********************F************************************F*********F*******************F***********F************************
"Mapa3":
*****************F**********************F************************************F*********F*******************F***********F************************

"Vida": 100

Aquests valors s'han d'afegir clicant a Insertar(a la barra horitzontal que hi ha adalt de la pàgina).
